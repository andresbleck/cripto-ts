Aplicación web para cotización de criptomonedas desarrollada en TypeScript, que consume una API REST para obtener valores en tiempo real. Implementa Zod para la validación y tipado seguro de los datos recibidos, garantizando mayor robustez y control de errores. El estado global de la aplicación se gestiona con Zustand, permitiendo una administración eficiente y simple de la información. La app permite consultar precios actualizados de distintas criptomonedas de forma dinámica y confiable.
